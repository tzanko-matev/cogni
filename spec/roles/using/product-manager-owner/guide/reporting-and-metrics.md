# Reporting and metrics

## What the reports show
- Pass rate per run and over time.
- Token usage and wall time trends.
- Per-task outcomes with citations.

## How to read trends
- Use `cogni report --range <start>..<end>` for history.
- Use `cogni compare` to see deltas for a specific change.

## Definitions
- Pass rate is the ratio of passing tasks to total tasks.
- A run is a benchmark execution at a specific commit.
