# Roadmap

## Near term

- MVP Go CLI with `init`, `validate`, `run`, `compare`, `report`.
- QA-only tasks with citation checks.
- Results and reports stored in configurable `output_dir`.
- Range-based trends via repo commit queries.
- OpenRouter provider and built-in agent.

## Medium term

- Multi-provider support (OpenAI, Anthropic).
- External agent adapters (Codex, Claude Code, Gemini).
- Multi-VCS support (jujutsu and others).
- Improved report UX and filtering.

## Long term

- Paid offering with team dashboards.
- Sandboxed runners for isolated benchmarking.
